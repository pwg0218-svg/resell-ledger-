# 서버 PC 설치 가이드

## 📋 USB에 담을 파일

`관리대장` 폴더 **전체**를 USB에 복사하세요.

---

## 🖥️ 서버 PC 설치 순서

### 1단계: Node.js 설치 (최초 1회)
1. https://nodejs.org 접속
2. **LTS 버전** 다운로드
3. 설치 (계속 Next 클릭)

### 2단계: 폴더 복사
USB에서 `관리대장` 폴더를 서버 PC로 복사
(예: `C:\서버\관리대장\`)

### 3단계: 서버 실행
**`서버시작.bat`** 더블클릭!

### 4단계: 접속
터미널에 나오는 **Network 주소**를 핸드폰에서 접속
(예: `http://192.168.0.100:5173`)

---

## 🔄 메인 PC에서 수정 후 업데이트 방법

1. 메인 PC에서 코드 수정
2. 수정된 파일을 서버 PC로 복사 (또는 전체 폴더 덮어쓰기)
3. 서버 PC에서 `서버시작.bat` 재실행

---

## ⚠️ 주의사항

- 검은 창 2개가 뜹니다 (**닫지 마세요!**)
- 모든 데이터는 `server/data.json`에 저장됩니다
